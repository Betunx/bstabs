# Canciones para Black Sheep

## Formato Requerido

Para cada canción necesitas:

```
Título:
Artista:
Tono: (ej: Bm, Am, C, etc.)
Tempo: (BPM)
Compás: (4/4, 3/4, 6/8, etc.)
Afinación: (Standard EADGBE, Drop D, etc.)
Dificultad: (beginner, intermediate, advanced)
```

## Tablatura

```
Intro:
e|-------------------|
B|-------------------|
G|-------------------|
D|-------------------|
A|-------------------|
E|-------------------|

Verso:

...
```
